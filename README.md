# Sweetheart Game Recommender
Project Idea
This is an interactive website that helps users choose a game based on their mood and preferences.
 Target Audience
This tool is for gamers who don’t know what game to play and feel overwhelmed by too many choices.
 Problem
There are too many games and people often struggle to decide what fits their mood.
Solution
This quiz asks users questions about their mood, focus, and play style, then recommends a game.
How it works
User answers 8 questions
Answers are stored using localStorage
A scoring system selects the best game
 Tools Used
HTML CSS JavaScript
Challenges
Fixing bugs with answers not matching
 Connecting all questions logically
Adjusting layout across pages

Future Improvements
Add more games
Show multiple recommendations
Improve design
